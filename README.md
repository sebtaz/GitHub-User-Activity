# GitHub-User-Activity
Fetches the recent activity of a GitHub user

To enter a user, you must enter it as an argument by typing the directory to run the file, then after typing the username.

Example:

PS C:\Users\sebta\Downloads\Github User Activity> & C:/Users/sebta/AppData/Local/Python/pythoncore-3.14-64/python.exe "c:/Users/sebta/Downloads/Github User Activity/Github User Activity.py" Sebtaz
